# Linux Bootstrap Scripts

- Distro: Debian
- Version: 11 "bullseye"

## Dotfiles

This will overwrite your current dotfiles.

```bash
$ cp -r dotfiles/. ~/
```

## Bash Scripts

You can run these bash scripts individually and
also inspect the code. These are all optional but
can be run on a clean Debian install.
